# About
## Server Application
A simple server application which is able to get some requests and able to respond to the requests.
This server expects requests form a To-do app mobile application.

## User Instructions 
Clone the github repo
```sh
git clone git@github.com:mohammed412anas/to-do-app-backend.git
```
Checkout to the directory
```sh
cd todobackend
```

Install all the dependencies
```sh
npm install
```

# Scripts Available
## npm run start
This script start the application as backend server, when started the application will be able to accepts requests from a client and respond to the requests accordingly.

## npm run test
This script runs the tests in the project

On default server runs at
```bash
http://localhost:10000/

```
